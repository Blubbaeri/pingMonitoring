Connection Quality Monitor Guide

About This Monitor

This is a pure client-side connection quality monitoring tool that measures network latency from your web browser to Google's connectivity testing servers.

Metric Definitions

  • Latency: The round-trip time (in milliseconds) it takes to reach the test server.
  • Jitter: The average difference in latency between consecutive probes. Lower values indicate a more stable connection.
  • Quality States: Excellent (<30ms) Good (<60ms) Fair (<100ms) Poor (≥100ms)
